# Nihongo Memory Coach - Stable Upgrade

Bản này bám theo file index.html người dùng đính kèm, giữ nguyên cấu trúc 1 file HTML để upload trực tiếp lên GitHub Pages.

## Cách dùng

1. Upload `index.html` lên repository GitHub.
2. Vào Settings → Pages.
3. Chọn Deploy from branch, branch `main`, folder `/root`.
4. Mở link GitHub Pages trên điện thoại.

## Ghi chú

- Tiến độ học được lưu trong localStorage của trình duyệt.
- Nên dùng nút “Xuất tiến độ” định kỳ để backup.
- Mazii được cấu hình theo hướng chỉ mở app Mazii, không tự mở web.
