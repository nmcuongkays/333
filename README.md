# Nihongo Memory Coach - Example Reading Clean

Base: bản đã thêm phiên âm dưới câu ví dụ.

Sửa trong bản này:
- Quay lại đúng base example-reading, không dùng bản external-linked.
- Sửa Bộ lọc nhanh/Tìm kiếm bằng chuẩn hóa Unicode: tìm được Kanji, kana, tiếng Việt có/không dấu.
- Rà soát lại raw trích PDF theo PDF gốc bằng PyMuPDF, thay raw lộn thứ tự bằng text lấy lại từ trang PDF.
- Sửa cách đọc bị thiếu okurigana do furigana nằm trên Kanji, ví dụ 蒸し暑い: むあつい -> むしあつい.
- Giữ nguyên Mazii app-only, SRS, Quiz, Drive Mode, dark mode và tab Dễ nhầm của base.
